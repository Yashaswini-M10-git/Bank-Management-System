# 🏦 Bank Management System

A simple Bank Management System developed using Python and Object-Oriented Programming (OOP).

## Features

- Create bank accounts
- Deposit money
- Withdraw money
- Check account balance
- Transfer money between accounts
- View transaction history
- Savings Account with interest calculation
- Current Account with overdraft facility
- Add, Search, Update, and Delete accounts using the Bank class

## OOP Concepts Used

- Classes and Objects
- Constructors (`__init__`)
- Encapsulation
- Inheritance
- Method Overriding
- super()
- Composition

## Project Structure

BankAccount
├── SavingsAccount
└── CurrentAccount

Bank


## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run:

```bash
python bank_management.py
```
